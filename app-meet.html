MEET App

Demo App Github Repo

MEET App is a cutting-edge, Progressive Web Application (PWA) designed to help users effortlessly discover and plan to attend tech events around the world. Built with the modern React framework, the app stands out by offering a seamless, offline-first experience, enabling users to access event information regardless of their internet connection status.

Features:

  • Discover Events by City: Users can search for tech events in specific cities, making it easier to find gatherings that match their interests and geographical preferences.
  • Dynamic Event Visualization: The app features interactive charts, including Event Genres and City Events charts, providing users with a visually appealing overview of events by category and location.
  • Real-time Event Updates: Leveraging the power of serverless functions, MEET App fetches real-time data, ensuring users have the latest information on tech events.
  • Interactive UI: With a focus on user experience, the app includes features like customizable event listings, detailed event information on demand, and the ability to save events for offline viewing.
  • PWA Capabilities: Designed as a Progressive Web App, MEET offers a native-like experience, including the ability to install the app directly on devices, receive push notifications, and access content offline.

Technologies Used:

  • React: For building a dynamic and responsive user interface.
  • Recharts: For generating interactive, responsive charts that enrich the user experience with data visualization.
  • Serverless Functions: To fetch and manage event data efficiently, providing scalability and reducing server-side overhead.
  • PWA Technology: Utilizing service workers, manifest files, and local caching to deliver an offline-first experience and native app-like features.

The MEET App project showcases my ability to develop modern web applications that are both functional and visually engaging. It demonstrates my proficiency with React, my understanding of PWA technologies, and my commitment to creating applications that provide real value to users.
